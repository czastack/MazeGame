作者：陈振安
时间：2016年5月
环境：Visual Studio 2015 (with Update 2)
依赖：本系统编译需要EasyX库

程序说明

main.cpp	主函数调用入口
Maze.cpp	迷宫类的实现，定义了迷宫的创建、初始化、生成地图、绘制迷宫、保存读取地图等函数
Maze.h	迷宫类的头文件
MazeGame.cpp	迷宫游戏核心控制类的实现，实现了系统初始化、游戏逻辑、用户界面交互、计算并显示最短路径等功能
MazeGame.h	迷宫游戏核心控制类的头文件
Menu.cpp	菜单模块的实现，定义菜单的存储结构和绘制函数
Menu.h	菜单模块的头文件
LinkStack.h	栈类的定义
